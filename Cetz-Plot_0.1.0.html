CeTZ-Plot
1 Introduction
CeTZ-Plot is a simple ploing library for use with CeTZ.
2 Usage
is is the minimal starting point:
#import "@preview/cetz:0.2.2"
#import "@preview/cetz-plot:0.1.0"
#cetz.canvas({
import cetz.draw: *
import cetz-plot: *
...
})
Note that plot functions are imported inside the scope of the canvas block. All following example code
is expected to be inside a canvas block, with the plot module imported into the namespace.
3 Plot
3.1.1 plot
Create a plot environment. Data to be ploed is given by passing it to the plot.add or other ploing
functions. e plot environment supports diﬀerent axis styles to draw, see its parameter axis-style:.



})
To draw elements insides a plot, using the plots coordinate system, use the plot.annotate(..) func-
tion.
